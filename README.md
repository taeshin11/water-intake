# Water Intake Calculator & Log

A free, beautiful, mobile-first web tool to calculate and track your daily hydration.

**Built by SPINAI**

## Features

- Personalized daily water intake calculation (weight, activity, climate)
- Interactive cup-grid dashboard — tap to track glasses consumed
- Animated progress bar with motivational messages
- LocalStorage streak tracker (7-day history)
- PWA-ready (installable to home screen)
- SEO-optimized for maximum organic traffic

## Stack

- HTML5 + Tailwind CSS (CDN) + Vanilla JS
- Hosted on Vercel (free tier)

## Deploy

```bash
npx vercel --prod
```
